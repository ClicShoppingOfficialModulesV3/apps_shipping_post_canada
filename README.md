# apps_shipping_post_canada

This apps allow you to display the shipping price come from Post Canada


 It's recommended to use this Apps with the shipping tracking Apps allow you to mangae some tracking system and display information for your customer :
https://www.clicshopping.org/forum/files/file/18-apps_configuration_shipping_status_tracking/
